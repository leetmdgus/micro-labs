**week02a 참조**

## **venv 생성**

```python
python -m venv venv
```

**PowerShell**

```
.\venv\Scripts\Activate.ps1
```

**lxml 설치**

```
.\venv\Scripts\python.exe -m pip install -U pip
.\venv\Scripts\python.exe -m pip install lxml
```



## 실행

```
.\venv\Scripts\python.exe main.py
```

```
index.html 실행
```
