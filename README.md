## 📘 개요  
`micro-labs`는 승현이 매주 진행하는 **기술 실험 프로젝트 모음집**입니다.  
짧게는 하루, 길게는 2~3일 동안 하나의 기술을 집중적으로 탐구하며,  
실제 서비스 개발 전에 **아이디어 검증 단계**로 사용됩니다.



---

## 🎯 목표  

- 매주 **하나의 기술 실험 → 결과 기록**
